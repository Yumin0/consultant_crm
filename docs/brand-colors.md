# 公司用色規範

> 來源：Yumin 提供的公司品牌色彩規範，2026-07-22 存入專案供之後選色時參考。
> 原則：所有 LINE Flex Message、網頁介面的顏色選擇，應優先使用主要色彩，輔助色彩用於需要多種顏色的場合，任何情況下主要色彩都應保持最高比重。

## 主要色彩

| 用途 | HEX |
|---|---|
| 品牌黃 | `#FFEA41` |
| 近黑 | `#111214` |
| 白 | `#FFFFFF` |

## 輔助色彩

| 說明 | HEX |
|---|---|
| 極淺灰 | `#F2F2F3` |
| 淺藍灰 | `#D4D8E3` |
| 中灰 | `#9599A4` |
| 淺藍 | `#ADCFFF` |
| 淺紫 | `#B9ADFF` |
| 淺薄荷綠 | `#C0EDE2` |

## 使用規範

為了在品牌傳播中實現一致的視覺效果，色彩的運用應以主要色彩系列為重點，並在適合的情況下輔以輔助色彩系列作為支持。

**不要使用不在上述清單中的顏色**（例如一般 Tailwind/Bootstrap 常見的鮮豔藍 `#1d4ed8`、`#2563eb` 等），這些不符合公司品牌規範。

## 衍生色階（漸層用）

需要「同一色系深淺漸層」時（例如卡片由深到淺排列），不要混用清單裡不同色相的色票（例如近黑＋中灰＋淺藍灰＋淺藍，會變成灰階混藍色、不是同一色系）。改成挑一個輔助色當基準，往「近黑 `#111214`」或「白 `#FFFFFF`」的方向內插出深/淺色階，維持同一色相。

範例：以輔助淺藍 `#ADCFFF` 為基準的 4 階漸層（用在企業主介紹卡）：

| 階層 | HEX | 說明 |
|---|---|---|
| 最深 | `#4F5E72` | `#ADCFFF` 與 `#111214` 約 4:6 混合 |
| 中深 | `#7E96B9` | `#ADCFFF` 與 `#111214` 約 7:3 混合 |
| 基準 | `#ADCFFF` | 規範原色 |
| 最淺 | `#CEE2FF` | `#ADCFFF` 與 `#FFFFFF` 約 6:4 混合 |

背景較淺（基準色以下）時標題文字要改用近黑 `#111214`，避免白字看不清楚。
